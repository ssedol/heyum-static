# 프로젝트 목적
- 고급 핸드메이드 액세서리 브랜드 홍보를 위한 1페이지(또는 미니멀) 정적 웹사이트
- 구매 기능 없이, 작품의 예술성과 가치를 보여주는 고품격 디지털 룩북 역할

# 타겟 유저
- 희소성 있고 고급스러운 커스텀 주얼리를 찾는 20-40대 고객

# 핵심 기능 및 페이지 구조
1. Hero Section: 브랜드 슬로건, 고화질 메인 배경 영상 또는 이미지, 미니멀한 내비게이션 바
2. Brand Story: 장인정신과 제작 과정을 녹여낸 감성적인 텍스트와 이미지 레이아웃
3. Gallery / Collection: 제품 라인업을 잡지 화보처럼 배치 (마우스 오버 시 부드러운 확대 효과)
4. Order Guide & Contact: 주문제작 프로세스 안내 및 카카오톡 채널/문자 예약 링크 버튼 (CTA)

# 디자인 요구사항 (중요)
- 전체적인 무드: 미니멀리즘, 여백의 미, 하이엔드 럭셔리
- 애니메이션: 스크롤 시 요소들이 부드럽게 나타나는 페이드인(Fade-in) 및 패닝 효과 적용
- 반응형 디자인: 모바일 화면에서도 고급스러운 레이아웃이 깨지지 않고 유지될 것

# 기술 스택
- HTML5, CSS3 (또는 Tailwind CSS), Vanilla JavaScript (정적 페이지 최적화)
